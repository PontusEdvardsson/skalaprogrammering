# Programmering 7 - losningsbibliotek

Detta repo innehaller enkla losningsforslag till programmeringsuppgifter i Python for arskurs 7.

Materialet ar byggt for att fungera som ett litet bibliotek av exempel som snabbt gar att visa pa lektion, dela med kollegor och anvanda som stod nar elever fastnar.

## Syfte

Filerna ar skrivna for att vara:

- enkla att lasa for nyborjare
- laetta att koppla till en specifik uppgift
- tillrackligt tydliga for gemensam genomgang i klassrummet
- anvandbara som losningsforslag, inte som enda mojliga facit

## Upplagg

- `.py` = korbara losningar i Python
- `.md` = svar eller forklaringar till uppgifter som framfor allt handlar om att lasa av, resonera eller forutse utskrift

Filnamnen foljer uppgiftsnumren i boken, till exempel:

- `01b_jag_programmerar_i_python.py`
- `14_triangelns_area.py`
- `27_miniraknare.py`

## Att anvanda pa lektion

- Lat eleverna prova uppgiften forst.
- Visa sedan en fil i taget som losningsforslag.
- Jamfor garna flera olika satt att skriva samma program.
- Utga fran att eleverna ar nyborjare och hall losningarna sa raka som mojligt.

## Notering

Losningarna ar skrivna med fokus pa tydlighet snarare an maximal "programmeringssmarthet". De ska vara laetta att forklara muntligt i klassrummet.
