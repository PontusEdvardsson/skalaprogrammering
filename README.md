# Skalaprogrammering

Detta repo innehaller enkla losningsforslag i Python for programmeringsuppgifter i hogstadiet.

Materialet ar gjort for att vara latt att dela med kollegor, visa pa lektion och anvanda som stod nar elever fastnar.

## Struktur

- `ak7/` = losningsbibliotek for arskurs 7
- `ak8/` = losningsbibliotek for arskurs 8
- `ak9/` = losningsbibliotek for arskurs 9

Varje arskurs ar uppdelad i undermappar efter vilken sida uppgifterna ligger pa i boken.

## Filtyper

- `.py` = korbara program i Python
- `.md` = svar, forklaringar eller resonemang till uppgifter som inte framfor allt ar en korbar kodlosning

## Att anvanda pa lektion

- Lat eleverna prova uppgiften sjalva forst.
- Visa sedan en enkel losning i taget.
- Jamfor garna flera mojliga satt att skriva samma program.
- Hall losningarna raka och begripliga for nyborjare.

## Notering

Losningarna ar skrivna med fokus pa tydlighet snarare an maximal programmeringssmarthet. De ska vara enkla att forklara muntligt i klassrummet.
